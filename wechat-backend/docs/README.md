## Structure Rules

- 新文档优先放到 `docs/` 下的主题目录
- 每个主题目录必须有一个 `README.md` 作为入口
- 每个主题目录优先拆成两类
  - `ops/`: 运维手册、部署、状态、手动触发、故障排查
  - `dev/`: 架构、设计、规则、路线图
- 主题文档只描述该主题，不在根目录继续堆放零散说明
